# imgui Lite

imgui lite is a modern version of the popular template; imgui with multiple features, it is written in CSS and HTML which makes it more accesable throughout the community

**Features:**

- Checkbox
- Button
- Text Input
- Number Input
- Dropdown Menu
- Toggle
- Dropdown Section
- Multiple Tabs
